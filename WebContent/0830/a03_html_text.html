1. 실행하자마자 #d1, #d2의 내용을 변경하기
	1) html() : 변경할 내용속에 html 태그가 포함되어 있을 경우
			     태그가 실행되어 나타난다.
		$('#d1').html("<h1>Hello</h1>");  ==>  Hello 만 출력
	2) text() : 변경할 내용속에 html 태그가 포함되어 있더라도
			     태그가 실행되지 않고 단순 글자로 표현된다. 
		$('#d2').text("<h1>Hello</h1>");  ==>  <h1>Hello</h1>가 출력
		
2. #d3, #d4 의 내용을 가져오기
	1) html() :  태그를 포함시켜 가져온다.
		alert($('#d3').html());	 ==>  #d3 안에 있는 태그까지 포함
									  안에 있는 태그가 여러개인경우 첫번째 인거만 태그를 포함하여 출력
	2) text() : 태그를 제외하고 가져온다.
		alert($('#d4').text());  ==>  #d4 안에 있는 글자만 출력
